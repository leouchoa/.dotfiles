# fclear

`fclear` cleans unsafe/spaced file names.

## Usage

```bash
fclear --help
fclear --dry-run "My File (1).JPG"
fclear --dir ~/Downloads
fclear --dir ~/Downloads --overwrite --rm-punct
```

## Completion

```bash
fclear completion zsh
fclear completion bash
fclear completion fish
```

Your `.zshrc` sources `fclear completion zsh` automatically when `fclear` is in `PATH`.
